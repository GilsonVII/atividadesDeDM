## 📂 O que tem na caixa? (Os Exercícios)

O repositório está dividido em 3 projetos separados. Cada pasta é um app independente!

### 1️⃣ Exercício 01: Captura e Alerta (`exercicio-01`)
Um app focado em interação básica. Ele captura o nome digitado pelo usuário usando um `TextInput` e, ao clicar no botão, exibe uma mensagem de boas-vindas na tela usando o componente `Alert`.

### 2️⃣ Exercício 02: Galeria com ScrollView (`exercicio-02`)
Focado em rolagem de tela. O app exibe uma lista de 6 imagens em sequência utilizando o componente `ScrollView`, permitindo que o usuário role a página para baixo para ver o conteúdo completo.

### 3️⃣ Exercício 03: Lista Dinâmica (`exercicio-03`)
A junção de tudo! Um app estilo "To-Do List" onde o usuário digita um item, clica no botão "Add" e o texto vai parar dentro de uma lista que cresce dinamicamente na tela, tudo isso gerenciado pelo hook `useState`.
